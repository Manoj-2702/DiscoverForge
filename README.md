## Kafka Setup
run these command in kafka folder
```bash
# start zookeeper
.\bin\windows\zookeeper-server-start.bart .\config\zookeeper.properties
```
```bash
# start kafka
.\bin\windows\kafka-server-start.bat .\config\server.properties
```
